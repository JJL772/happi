Command Line Utilities¶

happi command line tool

usage: happi [-h] [--path PATH] [--verbose] [--version]
             {search,add,edit,load} ...

Positional Arguments¶

cmd

Possible choices: search, add, edit, load

Subparsers to search, add, edit

Named Arguments¶

--path

path to happi configuration file

--verbose, -v

Show the degub logging stream

Default: False

--version, -V

Current version and location of Happi installation.

Default: False

Sub-commands:¶

search¶

Search the happi database

happi search [-h] search_criteria [search_criteria ...]

Positional Arguments¶

search_criteria

Search criteria: field=value. If field= is omitted, it will be assumed to be “name”. You may include as many search criteria as you like.

add¶

Add new entries

happi add [-h] [--clone CLONE]

Named Arguments¶

--clone

Name of device to use for default parameters

Default: “”

edit¶

Change existing entry

happi edit [-h] name edits [edits ...]

Positional Arguments¶

name

Device to edit

edits

Edits of the form field=value

load¶

Open IPython terminal with device loaded

happi load [-h] device_names [device_names ...]

Positional Arguments¶

device_names

Devices to load
